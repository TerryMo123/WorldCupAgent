# 世界杯球队对比分析 Skill

当用户请求两支球队胜负率或优劣势对比时，按以下流程执行。

## 分析流程

1. 调用 **stats** 工具获取两队近期战绩、胜率、历史交锋（H2H）
2. 调用 **rag** 工具检索两队战术文档与优劣势描述
3. 调用 **suggestion**（LLM）综合生成对比报告

## 输出格式（必须包含）

1. **数据摘要**：两队近 N 场胜率、进球/失球、FIFA 排名
2. **历史交锋**：H2H 胜平负与代表性比分
3. **优势对比**：各列 3 条，需标注来源（stats 或 rag）
4. **劣势对比**：各列 2-3 条
5. **综合判断**：给出倾向性结论，并说明置信度与不确定性
6. **免责声明**：预测仅供参考，非投注建议

## 约束

- 所有数字必须来自 stats 工具查询结果，禁止编造比分或胜率
- rag 检索不到的内容标注「暂无文档数据」
- 数据缺失时降级回答，不得虚构
- 用户未指定 `query` 时，默认做全面优劣对比

## 球队 ID 映射

用户可输入中文名或英文 id，系统会归一化为：`brazil`、`france`、`argentina`、`germany`
